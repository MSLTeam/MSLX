## 📖 项目简介

MSLX 是由 MSLTeam 开发的跨平台服务端管理工具。项目基于前后端分离架构（Vue + ASP.NET Core 10.0）构建。

它提供了一个简洁直观的 Web 远程管理界面，不仅能快速部署和管理各类 Minecraft 服务器，也支持托管任意通用的控制台程序。

## ✨ 核心特性

* **环境部署：** 内置下载镜像源，支持自动化下载与构建各类 MC Java 版核心（原版/插件/模组），同时兼容基岩版。
* **通用进程托管：** 不仅限于 Minecraft，可用于运行、管理及守护任意控制台程序。
* **自动化运维：** 提供灵活的定时任务、数据备份、实例状态监控与玩家管理功能。
* **内网穿透：** 原生集成 FRP 隧道，支持一键接入 MSLFrp、MEFrp 等服务，并开放高级自定义配置，解决公网联机问题。
* **跨平台支持：** 提供 Windows、Linux 预编译版本与 Docker 镜像，深度适配飞牛 fnOS 等现代化 NAS 系统。

## 🚀 快速开始

### 运行环境
* **支持系统：** Windows 10+ / Linux / Docker / NAS (注：Windows 7 可运行，但部分新版 Frpc 可能存在兼容性问题)
* **运行依赖：** `.NET 10.0`
