## 📖 项目简介

**MSLX** 是由 MSL 原班团队 (MSLTeam) 倾力打造的新一代服务端管理工具。基于前后端分离架构（Vue + ASP.NET Core 10.0）开发。

它不仅传承了 MSL 经典的零门槛 UI 设计语言，更带来了真正的跨平台运行能力与强大的远程 Web 管理体验。无论是搭建各类 Minecraft 服务器，还是托管通用的控制台程序，MSLX 都能为您提供极度流畅的运维体验。

## ✨ 核心特性

* 🎮 **极速环境部署：** 原生支持 MC Java 版（原版/插件/模组核心）高速下载与自动化构建（内置镜像源），并兼容基岩版服务端。
* 🚀 **通用进程托管：** 打破业务局限，支持运行、管理并实时守护**任意控制台程序**。
* 🛠️ **专业运维矩阵：** 集成灵活的定时任务、自动化数据备份、玩家管理与实例状态监控，让复杂运维化繁为简。
* 🌍 **无缝内网穿透：** 原生集成 FRP 隧道，一键接入 MSLFrp、MEFrp 等优质网络服务商，支持高级自定义配置，轻松跨越公网限制。
* 📦 **全平台无缝运行：** 完美支持 Windows、Linux，并提供完善的 Docker 镜像，深度适配飞牛 fnOS 现代化 NAS 私有云系统。

## 🚀 快速开始

### 运行环境
* **支持系统：** Windows 10 及以上 / Linux / Docker / 现代化 NAS (注：Windows 7 可运行，但可能存在新版 Frpc 兼容性问题)
* **底层框架：** `.NET 10.0`
