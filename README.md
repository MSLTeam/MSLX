# Minecraft Server Launcher X (MSLX)
![GitHub Release](https://img.shields.io/github/v/release/MSLTeam/MSLX)
![GitHub License](https://img.shields.io/github/license/MSLTeam/MSLX)
![GitHub Downloads (all assets, all releases)](https://img.shields.io/github/downloads/MSLTeam/MSLX/total)
![GitHub Issues or Pull Requests](https://img.shields.io/github/issues/MSLTeam/MSLX)
![GitHub Repo stars](https://img.shields.io/github/stars/MSLTeam/MSLX)

MSLX是由MSL原班团队MSLTeam倾力打造的全新一代开服工具，基于.NET Core 10.0环境开发 。它传承了MSL经典的UI设计语言，旨在让操作零门槛，完美支持跨平台运行，并引入了强大的远程访问功能。
